#  MVVM Task Manager

A simple SwiftUI-based task management app that helps you organize and track your tasks. Tasks can be created, completed, or deleted with intuitive gestures.

## âœ¨ Features

- **Add new tasks** with a title using an alert-style input  
- **Swipe to complete or delete** tasks directly from the list  
- **Separate sections** for *Waiting* and *Completed* tasks  
- **Toggle completion state** with a single gesture  
- **MVVM architecture** for clean separation of logic and UI
